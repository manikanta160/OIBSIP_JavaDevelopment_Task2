# Smart Number Guessing Game

## Description
A console-based Smart Number Guessing Game developed in Java using Object-Oriented Programming (OOP) concepts. This project was created as part of the Oasis Infobyte Java Development Internship (Task 2).

## Features
- Menu-driven program
- Easy, Medium, and Hard difficulty levels
- Random number generation
- Score calculation
- Player statistics
- Multiple rounds of gameplay
- Input validation

## Technologies Used
- Java
- IntelliJ IDEA

## Project Structure
- Main.java
- Menu.java
- Game.java
- Player.java
- Difficulty.java
- ScoreCalculator.java

## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
4. Follow the on-screen instructions.
